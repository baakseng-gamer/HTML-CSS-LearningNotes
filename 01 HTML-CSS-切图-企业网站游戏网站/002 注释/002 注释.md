# 002 注释

视频序号009



**写法：**

```
<!-- 注释的内容 -->
```

在浏览器中看不到，只能在代码中看到注释的内容。

**意义：**

1. 把暂时不用的代码注释起来，方便以后使用。
2. 对开发人员进行提示。

**VSCODE快捷进行注释：**

1. ctrl + /   整行注释

2. shift + alt + a  块注释

示例：

```
<body>
    <!-- hello world -->
    <div>abc</div>  6666
</body>
```

运行结果：

页面上就显示两行，分别是 abc 和 666，而 hello world 没被显示。
