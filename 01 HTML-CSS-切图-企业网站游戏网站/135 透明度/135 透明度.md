# 1. 透明度示例

```
opacity : 0;
```

取值范围：

* 0(透明) ~ 1(不透明)

* 0.5(半透明)



```
..... : rgba()
```

a 在 0 ~ 1区间内，例如：rgba(255, 100, 102, 0.5)。



**==注意==**

* 可以让指定的样式透明，而不影响其他样式。
* 占空间的、所有的子内容也会透明。



示例：

```
        .box1{
            width: 100px;
            height: 100px;
            background: tomato;
            opacity: 0.5;
        }
        .box2{
            width: 100px;
            height: 100px;
            background: tomato;
            opacity: 0;
            /* display: none; */
            /* visibility: hidden; */
        }
        .box3{
            width: 100px;
            height: 100px;
            background: tomato;
        }
```
